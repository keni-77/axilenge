# aɕilengə
## 文
[主語] `ɕi` [述語] ( `we` [直接目的語] `te` [間接目的語])  

## 修飾
[形容詞（連体形）] [名詞]  
[形容詞（連用形）] [用言（動詞/形容詞）]  

# 名詞
## 文法
名詞は基本的に `ə` で終わるが、人称代名詞は例外  
複数形は例外なくしっぽに `n` を付ける  

## 人称代名詞
一人称 `a`  
二人称 `u`  
彼 `e`  
彼女 `i`  
その方（性別不明） `o`

### 前置詞
#### 文法
名詞または節を後ろに置いて使い、様態は終止形としても使える  
連用形に活用したときは、修飾する語より後につける  

Aのような（様態） `janu A`  
Aのような（例示） `gazu A`  
Aと一緒の `tagu A`  
Aのおかげの `alu A`  
Aによる、Aを使った `jasu A`
