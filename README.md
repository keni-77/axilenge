# 名詞
## 文法
名詞は基本的に `ə` で終わるが、人称代名詞は例外  
複数形は例外なくしっぽに `n` を付ける  

## 人称代名詞
一人称 `a`  
二人称 `u`  
彼 `e`  
彼女 `i`  
それ・そいつ `o`
